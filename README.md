# 摸摸鱼阅读

一款 macOS 端轻量级小说阅读器，专为"摸摸鱼"设计 —— 悬浮窗置顶 + 鼠标悬停显示文字，让你在工作时也能安心看小说。

## 功能特性

- **悬浮窗置顶** — 无边框悬浮窗口，始终显示在最上层，不影响其他工作
- **悬停显示** — 鼠标悬停时才显示文字，移开后自动隐藏，摸鱼必备
- **书库管理** — 导入、删除书籍，记录上次阅读时间，支持批量导入
- **章节导航** — 自动识别中文章节目录（第X章/回/节/卷），一键跳转
- **阅读进度** — 自动保存阅读位置，下次打开自动恢复
- **书签系统** — 随时添加书签，快速跳转到标记位置
- **编码兼容** — 自动检测 UTF-8、UTF-16、GBK/GB18030 编码
- **个性化设置** — 自定义字体、字号、颜色、背景、透明度、窗口大小

## 系统要求

- macOS 13.0 (Ventura) 或更高版本
- 支持 `.txt` 格式文件

## 安装方式

### 方式一：下载安装包（推荐）

1. 从 [Releases](https://github.com/ChenQqing/MoMoYu-Reader/releases) 页面下载 `MoMoYu-Reader.dmg`
2. 双击打开 DMG，将 `摸摸鱼.app` 拖到 `应用程序` 文件夹
3. **首次打开前**，在终端运行以下命令解除系统限制：

```bash
sudo xattr -r -d com.apple.quarantine /Applications/摸摸鱼.app
```

> 本应用未付费签名，macOS 会标记为"已损坏"，运行上述命令即可正常使用。

### 方式二：源码编译

```bash
git clone https://github.com/ChenQqing/MoMoYu-Reader.git
cd MoMoYu-Reader/NovelReader
open NovelReader.xcodeproj
```

在 Xcode 中选择 "My Mac"，点击运行即可（源码编译无需上述命令）。

## 使用说明

1. 启动后进入书库界面，点击 `+` 按钮或双击空白区域添加 `.txt` 文件
2. 点击书籍即可开始阅读
3. 右键菜单可访问：目录、书签、设置等功能

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd + O` | 打开文件 |
| `Cmd + ,` | 打开设置 |

### 设置选项

| 设置项 | 说明 |
|--------|------|
| 文字颜色 | 自定义阅读文字颜色 |
| 背景颜色 | 自定义窗口背景颜色 |
| 背景透明度 | 调节背景透明程度（0-100%） |
| 字体 | 选择系统已安装的字体 |
| 字号 | 12-36pt 可调 |
| 行间距 | 1.0-3.0 可调 |
| 悬停显示 | 开启/关闭鼠标悬停才显示文字 |
| 窗口大小 | 宽度 250-1200px，高度 300-1000px |
