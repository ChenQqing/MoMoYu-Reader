# NovelReader - macOS 小说悬浮阅读器设计文档

## 概述

一款 macOS 原生小说阅读应用，支持悬浮窗显示小说内容，可在任何软件之上阅读。支持背景自定义、全透明模式、鼠标悬停显隐文字等功能。

## 技术选型

- **语言**: Swift 5.9+
- **UI 框架**: SwiftUI + AppKit 混合
- **最低系统**: macOS 13 (Ventura)
- **构建工具**: Xcode 项目

## 架构

```
NovelReaderApp (SwiftUI App Entry)
├── FileManagerService → 负责 .txt 文件读取和编码检测
├── ReadingView → 主阅读悬浮窗口
│   ├── 文本显示区（ScrollView + 分页）
│   ├── 悬停检测（NSTrackingArea）
│   └── 窗口控制（大小、拖拽）
├── SettingsView → 独立设置窗口
│   ├── 背景颜色选择器
│   ├── 字体设置（大小、类型、行间距）
│   ├── 透明度设置
│   └── 窗口大小设置
├── ReadingStateManager → 阅读状态管理
│   ├── 书签/阅读位置持久化
│   └── UserDefaults + JSON 存储
└── HoverManager → 悬停显隐逻辑
```

## 核心功能

### 1. 文件选择

- 启动时弹出 `NSOpenPanel`，限定 `.txt` 文件
- 支持 UTF-8、GBK 编码自动检测
- 读取文件内容并按段落分割

### 2. 悬浮阅读窗口

- `NSWindow` level 设为 `.floating`，始终置顶
- 支持自由拖拽（标题栏或内容区拖拽）
- 支持调整大小（右下角拖拽）
- 背景支持三种模式：
  - 纯色背景（用户选择颜色）
  - 半透明背景（颜色 + 透明度滑块）
  - 全透明背景（透明度 0，仅显示文字）
- 窗口圆角可调

### 3. 悬停显隐功能

- 使用 `NSTrackingArea` 监听鼠标进出事件
- 鼠标移开：文字完全隐藏（opacity = 0），只剩背景
- 鼠标进入：文字渐显（0.2s 动画过渡）
- 该功能可开关（设置中切换）

### 4. 阅读功能

- **滚动模式**: 连续滚动阅读
- **分页模式**: 按页翻阅，支持键盘左右箭头翻页
- 自动保存阅读位置（基于字符偏移量）
- 支持手动添加书签

### 5. 字体设置

- 字体大小：12pt - 36pt
- 字体类型：系统字体列表选择
- 行间距：1.0 - 3.0 倍
- 文字颜色可自定义

### 6. 设置窗口

- 独立 `NSWindow`，从阅读窗口点击齿轮按钮打开
- 所有设置实时预览
- 设置项通过 UserDefaults 持久化

## 数据持久化

| 数据 | 存储方式 | 位置 |
|------|----------|------|
| 阅读位置 | UserDefaults | app sandbox |
| 书签列表 | JSON 文件 | ~/Library/Application Support/NovelReader/ |
| 应用设置 | UserDefaults | app sandbox |
| 最近打开文件 | UserDefaults | app sandbox |

## 窗口规格

### 阅读窗口

- 默认大小：400x600
- 最小大小：250x300
- 默认背景：半透明深灰 (r:0.15, g:0.15, b:0.15, a:0.85)
- 默认字体：System 16pt
- 默认行间距：1.5

### 设置窗口

- 固定大小：350x450
- 非置顶，跟随应用窗口层级

## 交互流程

```
启动应用
  ↓
弹出文件选择器 (NSOpenPanel)
  ↓
选择 .txt 文件
  ↓
打开悬浮阅读窗口（自动恢复上次阅读位置）
  ↓
用户可随时：
  - 点击齿轮 → 打开设置窗口
  - 拖拽窗口 → 移动位置
  - 拖拽边缘 → 调整大小
  - 鼠标移出 → 文字隐藏（如开启该功能）
  - 滚动/翻页 → 阅读内容
  - Cmd+W → 关闭阅读窗口
  - Cmd+Q → 退出应用
```

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| Cmd+O | 打开新文件 |
| Cmd+, | 打开设置 |
| Cmd+W | 关闭阅读窗口 |
| Cmd+Q | 退出应用 |
| ← / → | 分页模式下翻页 |
| Space | 分页模式下下一页 |
